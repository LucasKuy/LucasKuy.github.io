Fuel Costing Web Application for Wessuc Inc.

Python and Flask application that helps accountants calculate fuel costs for trucks and projects.

Truck Repair Scheduling Automation

Workflow that connects Grist, Zoho Flow, email, and calendars to keep repair schedules up to date.

  • Connected the Grist repair tracking system to Zoho Flow using webhooks so changes could trigger actions.
  • Sent emails, Zoho Cliq messages, and calendar events when repair statuses changed or appointments moved.
  • Helped mechanics, supervisors, and office staff see the latest information in one place.

Technologies: Zoho Flow, Grist, webhooks, Zoho Cliq, calendar services

Camera Photo Logging Application

Technologies: Zoho Creator, Zoho WorkDrive, Deluge, scheduled tasks, email integration
